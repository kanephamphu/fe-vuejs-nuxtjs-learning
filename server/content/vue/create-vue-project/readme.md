# Create Vue Project

The recommended way to start a new Vue project is using the official `create-vue` tool.

## Steps
1. Open your terminal.
2. Run the command: `npm create vue@latest`
3. Follow the prompts (Select TypeScript, Vue Router, Pinia, etc.).
4. Initialize and run:
   ```bash
   cd <project-name>
   npm install
   npm run dev
   ```

## Project Structure
- `src/main.js`: The entry point.
- `src/App.vue`: The root component.
- `src/components/`: Where your components live.
- `public/`: Static assets.

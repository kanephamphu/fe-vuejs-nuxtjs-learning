## Practice: Your First Vue App

Let's build a simple counter to understand Declarative Rendering and Reactivity.

In the editor below, we have a basic Vue setup.
1. Return an object from the `setup()` function.
2. Use `ref()` to create reactive state.
3. Bind the state to the template using mustaches `{{ }}`.

### Tasks:
1. Define a `message` variable using `ref('Hello Vue!')`.
2. Define a `counter` variable initialized to `0`.
3. Create a function `increment` that adds 1 to `counter.value`.
4. Return them all from the `setup` function.

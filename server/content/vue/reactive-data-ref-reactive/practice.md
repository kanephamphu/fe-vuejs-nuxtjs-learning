## Practice: Ref vs Reactive

Experiment with both types of reactivity.

### Tasks:
1. Import `ref` and `reactive` from 'vue'.
2. Create a `count` using `ref(0)`.
3. Create a `user` object using `reactive({ name: 'Alice' })`.
4. Create a function `update` that:
   - Increments `count`.
   - Changes `user.name` to 'Bob'.

# Create Nuxt App

Starting a new Nuxt project is easy with `nuxi`, the Nuxt CLI.

## Steps
1. Open your terminal.
2. Run: `npx nuxi@latest init <project-name>`
3. Choose your preferred package manager (npm, pnpm, yarn).
4. Navigate and run:
   ```bash
   cd <project-name>
   npm run dev
   ```

## nuxi CLI
`nuxi` can also be used to generate files:
- `npx nuxi add page about`: Creates `pages/about.vue`.
- `npx nuxi add component Header`: Creates `components/Header.vue`.

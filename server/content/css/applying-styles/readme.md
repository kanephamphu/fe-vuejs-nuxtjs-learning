# Applying Styles

There are three ways to apply CSS to your HTML.

## 1. Inline Styles
Applied directly to the element using the `style` attribute.
```html
<h1 style="color: blue;">Hello</h1>
```

## 2. Internal Styles
Defined inside a `<style>` tag in the `<head>` of your HTML document.

## 3. External Styles (Best Practice)
Defined in a separate `.css` file and linked in the `<head>`.
```html
<link rel="stylesheet" href="style.css">
```

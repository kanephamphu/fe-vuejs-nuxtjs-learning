- `justify-content`: Aligns items on the main axis (start, center, space-between).
- `align-items`: Aligns items on the cross axis (stretch, center, baseline).

## Flex Item Properties
- `flex-grow`: Ability for an item to grow.
- `flex-shrink`: Ability for an item to shrink.
- `flex-basis`: Default size before remaining space is distributed.

```css
.container {
  display: flex;
  justify-content: center;
  align-items: center;
  height: 100vh;
}
```

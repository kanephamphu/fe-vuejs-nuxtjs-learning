# Colors in CSS

CSS allows you to specify colors for text, backgrounds, borders, and more.

## Color Formats
1. **Names**: `red`, `blue`, `transparent`
2. **Hex**: `#ff0000` (Red), `#00ff00` (Green)
3. **RGB/RGBA**: `rgb(255, 0, 0)`, `rgba(0, 0, 255, 0.5)` (semi-transparent)
4. **HSL**: `hsl(0, 100%, 50%)` (Hue, Saturation, Lightness)

```css
body {
  background-color: #f4f4f4;
  color: hsl(200, 50%, 20%);
}
```

# Defining Layout in CSS
Layout controls how elements are arranged on the page.

## Display Property
The core of layout. Common values:
- `block`: Takes full width, starts on new line (e.g., `div`, `p`).
- `inline`: Takes only necessary width, sits on same line (e.g., `span`, `a`).
- `inline-block`: Inline flow but respects width/height.
- `flex`: Flexible Box Layout.
- `grid`: Grid Layout.

## Box Model
Every element is a rectangular box consisting of:
1. **Content**
2. **Padding** (inner space)
3. **Border**
4. **Margin** (outer space)

# Class Selectors

Class selectors allow you to target multiple elements with the same style.

## Syntax
In CSS, classes start with a dot `.`.
```css
.highlight {
  background-color: yellow;
}
```

In HTML, you use the `class` attribute.
```html
<p class="highlight">This is highlighted.</p>
<span class="highlight">This too!</span>
```

## Why use Classes?
Classes are reusable and allow for flexible styling without affecting every tag of a certain type.

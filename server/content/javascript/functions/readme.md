# Functions

Functions are reusable blocks of code that perform a specific task.

## Function Declaration
```javascript
function greet(name) {
  return "Hello " + name;
}
```

## Function Expression
```javascript
const add = function(a, b) {
  return a + b;
};
```

## Arrow Functions (ES6)
```javascript
const multiply = (a, b) => a * b;
```

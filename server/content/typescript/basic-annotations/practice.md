# Practice: Typed Variables

Try creating some typed variables in a `.ts` file.

1. Create a variable `age` as a `number`.
2. Create an array `skills` as an array of `strings`.
3. Create a function `add` that takes two `numbers` and returns a `number`.

```typescript
// Your code here
const age: number = 25;
const skills: string[] = ["Vue", "Nuxt", "TypeScript"];

function add(a: number, b: number): number {
  return a + b;
}
```
